# 💻 Chirag Computers Website

A full-stack business website for **Chirag Computers (Visnagar, Gujarat)** to showcase laptop sales, repair services, and customer reviews.

---

## 🌐 Live Website

👉 https://chirag-computer.onrender.com/

---

## 🚀 Features

* 🛒 Display second-hand laptops with details
* 🛠️ Laptop repair services section
* 🔐 Admin-only product upload system
* 📸 Image upload for products
* ⭐ Customer reviews system
* 🎥 YouTube videos for trust & verification
* 📱 Responsive design

---

## 🧰 Tech Stack

* Backend: Python (Flask)
* Frontend: HTML, CSS
* Database: SQLite
* Hosting: Render

## ⚙️ How to Run Locally

```bash
git clone https://github.com/YOUR-USERNAME/chirag-computers.git
cd chirag-computers

pip install -r requirements.txt
python app.py
```

Open browser:
http://127.0.0.1:5000
